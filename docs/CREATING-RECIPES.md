# 📝 Creando Recetas Personalizadas

Las recetas son plantillas que guían a Copilot para realizar tareas específicas en tu proyecto SICEM.

## 📂 Ubicación

Las recetas deben estar en tu repositorio SICEM en: .copilot/recipes/

## 🔄 Cómo Funcionan

1. Bot lee receta desde workspace/sicem/.copilot/recipes/
2. Personaliza el prompt con tu solicitud
3. Copilot ejecuta los cambios
4. Commits automáticos a GitHub

## ⚡ Ventajas

✅ Más rápido (sin HTTP)
✅ Más confiable  
✅ Siempre sincronizado
✅ Fácil de editar

Ver SETUP-AUTO-EXECUTION.md para más detalles.
